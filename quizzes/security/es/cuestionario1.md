# 🔒 Security - Cuestionario 1

## Preguntas

### 1. 🔒 ¿Qué es la ciberseguridad? 🟢

A) 🔒 Protección de sistemas digitales contra amenazas

B) 🔧 Servicio de reparación de computadoras

C) 📦 Instalación de software

D) 🌐 Navegación por internet

**Respuesta correcta**: A) 🔒 Protección de sistemas digitales contra amenazas

> 💡 La ciberseguridad defiende computadoras, servidores, redes y datos de ataques maliciosos. ¡Como un guardia de seguridad digital!

### 2. 🛡️ ¿Qué es la autenticación? 🟢

A) 🛡️ Verificar la identidad del usuario

B) 🔧 Autorización del sistema

C) 📦 Cifrado de datos

D) 🌐 Monitoreo de red

**Respuesta correcta**: A) 🛡️ Verificar la identidad del usuario

> ⚡ La autenticación confirma que eres quien dices ser. ¡Como mostrar tu ID en un edificio seguro!

### 3. 🔑 ¿Qué es la autorización? 🟢

A) 🔑 Otorgar permisos de acceso a usuarios autenticados

B) 🔧 Proceso de registro de usuario

C) 📦 Creación de contraseñas

D) 🌐 Verificación de inicio de sesión

**Respuesta correcta**: A) 🔑 Otorgar permisos de acceso a usuarios autenticados

> 📘 La autorización determina qué pueden hacer los usuarios autenticados. ¡Como obtener una tarjeta con acceso específico al edificio!

### 4. 🔐 ¿Qué es el cifrado? 🟢

A) 🔐 Convertir datos en formato ilegible

B) 🔧 Compresión de datos

C) 📦 Almacenamiento de archivos

D) 🌐 Transmisión de datos

**Respuesta correcta**: A) 🔐 Convertir datos en formato ilegible

> 🎯 El cifrado codifica datos para que solo las partes autorizadas puedan leerlos. ¡Como escribir en código secreto!

### 5. 🕷️ ¿Qué es una vulnerabilidad? 🟡

A) 🕷️ Debilidad de seguridad que puede ser explotada

B) 🔧 Error del sistema

C) 📦 Bug de software

D) 🌐 Problema de red

**Respuesta correcta**: A) 🕷️ Debilidad de seguridad que puede ser explotada

> ⚡ Las vulnerabilidades son fallas de seguridad que los atacantes pueden explotar. ¡Como una grieta en tu armadura digital!

### 6. 🎯 ¿Qué es una amenaza? 🟡

A) 🎯 Peligro potencial para sistemas de información

B) 🔧 Advertencia del sistema

C) 📦 Mensaje de error

D) 🌐 Alerta de red

**Respuesta correcta**: A) 🎯 Peligro potencial para sistemas de información

> 💡 Las amenazas son ataques potenciales o eventos dañinos. ¡Como nubes de tormenta en el horizonte!

### 7. 🚨 ¿Qué es malware? 🟢

A) 🚨 Software malicioso diseñado para dañar sistemas

B) 🔧 Herramienta de mantenimiento del sistema

C) 📦 Organizador de archivos

D) 🌐 Utilidad de red

**Respuesta correcta**: A) 🚨 Software malicioso diseñado para dañar sistemas

> 📘 El malware incluye virus, trojanos, ransomware, spyware. ¡Tus enemigos digitales!

### 8. 🎣 ¿Qué es phishing? 🟡

A) 🎣 Intentos engañosos de robar información sensible

B) 🔧 Pesca de datos en la red

C) 📦 Gestión de correo electrónico

D) 🌐 Desarrollo de sitios web

**Respuesta correcta**: A) 🎣 Intentos engañosos de robar información sensible

> 🚨 El phishing usa correos/sitios falsos para engañar a usuarios y revelar credenciales. ¡Como artistas de la estafa digital!

### 9. 🔥 ¿Qué es un firewall? 🟢

A) 🔥 Barrera de seguridad de red que controla el tráfico

B) 🔧 Sistema de supresión de incendios

C) 📦 Respaldo del sistema

D) 🌐 Bloqueador de internet

**Respuesta correcta**: A) 🔥 Barrera de seguridad de red que controla el tráfico

> 🎯 Los firewalls monitorean y controlan el tráfico de red entrante/saliente. ¡Tu portero digital!

### 10. 🔍 ¿Qué es software antivirus? 🟢

A) 🔍 Programa que detecta y elimina malware

B) 🔧 Optimizador de sistema

C) 📦 Limpiador de archivos

D) 🌐 Escáner de red

**Respuesta correcta**: A) 🔍 Programa que detecta y elimina malware

> ⚡ El software antivirus protege contra software malicioso. ¡Como tener un sistema inmune digital!

### 11. 🔐 ¿Qué es la autenticación de dos factores (2FA)? 🟡

A) 🔐 Seguridad usando dos métodos de verificación diferentes

B) 🔧 Sistema de doble contraseña

C) 📦 Inicio de sesión de dos pasos

D) 🌐 Acceso de red dual

**Respuesta correcta**: A) 🔐 Seguridad usando dos métodos de verificación diferentes

> 💡 2FA combina algo que sabes (contraseña) con algo que tienes (teléfono). ¡Doble seguridad!

### 12. 🗝️ ¿Qué es una VPN? 🟡

A) 🗝️ Red Privada Virtual para conexiones seguras

B) 🔧 Red Muy Privada

C) 📦 Red Pública Virtual

D) 🌐 Red Privada Verificada

**Respuesta correcta**: A) 🗝️ Red Privada Virtual para conexiones seguras

> 📘 Las VPN crean túneles cifrados para navegación segura en internet. ¡Como tener una autopista privada en internet!

### 13. 🔒 ¿Qué es SSL/TLS? 🟡

A) 🔒 Protocolos de cifrado para comunicación web segura

B) 🔧 Especificaciones de lenguaje del sistema

C) 📦 Estándares de licencia de software

D) 🌐 Servicios de ubicación del servidor

**Respuesta correcta**: A) 🔒 Protocolos de cifrado para comunicación web segura

> 🚀 SSL/TLS cifra datos entre navegadores y servidores. ¡Busca el icono de candado en tu navegador!

### 14. 🕳️ ¿Qué es un ataque de día cero? 🔴

A) 🕳️ Ataque que explota vulnerabilidades desconocidas

B) 🔧 Ataque del primer día del sistema

C) 📦 Infección inicial de malware

D) 🌐 Brecha inmediata de red

**Respuesta correcta**: A) 🕳️ Ataque que explota vulnerabilidades desconocidas

> ⚡ Los ataques de día cero usan vulnerabilidades desconocidas para los proveedores. ¡Como ser atacado con un arma secreta!

### 15. 🔍 ¿Qué son las pruebas de penetración? 🔴

A) 🔍 Pruebas autorizadas para encontrar debilidades de seguridad

B) 🔧 Pruebas de rendimiento del sistema

C) 📦 Análisis de penetración de software

D) 🌐 Pruebas de velocidad de red

**Respuesta correcta**: A) 🔍 Pruebas autorizadas para encontrar debilidades de seguridad

> 🎯 Las pruebas de penetración simulan ataques para encontrar vulnerabilidades. ¡Como contratar hackers amigables para probar tus defensas!

### 16. 🏰 ¿Qué es una DMZ en seguridad de red? 🔴

A) 🏰 Zona Desmilitarizada - red intermedia entre interna y externa

B) 🔧 Zona de Gestión de Datos

C) 📦 Zona de Medios Digitales

D) 🌐 Zona de Memoria Directa

**Respuesta correcta**: A) 🏰 Zona Desmilitarizada - red intermedia entre interna y externa

> 💡 La DMZ aísla servicios públicos de redes internas. ¡Como tener un vestíbulo antes de las áreas seguras!

### 17. 🔑 ¿Qué es PKI? 🔴

A) 🔑 Infraestructura de Clave Pública para gestionar certificados digitales

B) 🔧 Integración de Clave Privada

C) 📦 Identificación de Clave Pública

D) 🌐 Interfaz de Clave Protegida

**Respuesta correcta**: A) 🔑 Infraestructura de Clave Pública para gestionar certificados digitales

> 📘 PKI gestiona certificados digitales y claves de cifrado. ¡La base de la seguridad web moderna!

### 18. 🚨 ¿Qué es un ataque DDoS? 🟡

A) 🚨 Denegación de Servicio Distribuida - saturar servidores con tráfico

B) 🔧 Datos Directos de Servicio

C) 📦 Datos Distribuidos sobre Servicio

D) 🌐 Denegación Dinámica de Sistemas

**Respuesta correcta**: A) 🚨 Denegación de Servicio Distribuida - saturar servidores con tráfico

> 🌊 Los ataques DDoS inundan servidores con tráfico para hacerlos no disponibles. ¡Como un atasco de tráfico digital!

### 19. 🔐 ¿Qué es el cifrado en reposo? 🔴

A) 🔐 Cifrar datos almacenados en discos/bases de datos

B) 🔧 Cifrar sistemas inactivos

C) 📦 Cifrar archivos de respaldo

D) 🌐 Cifrar redes desconectadas

**Respuesta correcta**: A) 🔐 Cifrar datos almacenados en discos/bases de datos

> ⚡ El cifrado en reposo protege datos cuando están almacenados. ¡Como guardar tus objetos de valor en una caja fuerte!

### 20. 🌐 ¿Qué es el cifrado en tránsito? 🔴

A) 🌐 Cifrar datos mientras están siendo transmitidos

B) 🔧 Cifrar sistemas en movimiento

C) 📦 Cifrar archivos transferidos

D) 🔒 Cifrar datos temporales

**Respuesta correcta**: A) 🌐 Cifrar datos mientras están siendo transmitidos

> 🚚 El cifrado en tránsito protege datos mientras se mueven entre sistemas. ¡Como un camión blindado para tus datos!

### 21. 🔍 ¿Qué es el principio de menor privilegio? 🔴

A) 🔍 Otorgar derechos mínimos de acceso necesarios para funciones laborales

B) 🔧 Reducir privilegios del sistema

C) 📦 Limitar permisos de software

D) 🌐 Minimizar acceso de red

**Respuesta correcta**: A) 🔍 Otorgar derechos mínimos de acceso necesarios para funciones laborales

> 🎯 El menor privilegio reduce la superficie de ataque limitando el acceso. ¡Como dar a las personas solo las llaves que realmente necesitan!
