# 🗄️ Databases - Questions 1

## Questions

### 1. 🗄️ What is a database? 🟢

A) 🗄️ System to store and manage data

B) 🔧 File server

C) 📦 Package manager

D) 🌐 Network system

**Correct Answer**: A) 🗄️ System to store and manage data

> 💡 A database is an organized system for storing, retrieving, and managing information. Like a well-organized digital library!

### 2. 🔍 What is SQL? 🟢

A) 🔍 Structured Query Language

B) 🔧 File system

C) 📦 Network protocol

D) 🌐 Programming language

**Correct Answer**: A) 🔍 Structured Query Language

> 📘 SQL is the standard language for interacting with relational databases. Like the universal language for talking to databases!

### 3. 🔗 What is a relational database? 🟢

A) 🔗 Database that organizes data in related tables

B) 🔧 Cloud database

C) 📦 File database

D) 🌐 Distributed database

**Correct Answer**: A) 🔗 Database that organizes data in related tables

> ⚡ Relational databases use tables to organize data with relationships between them. Like a well-organized folder system!

### 4. 🔑 What is a primary key? 🟡

A) 🔑 Unique field that identifies each record

B) 🔒 Access password

C) 🛡️ Encryption key

D) 🔐 Security key

**Correct Answer**: A) 🔑 Unique field that identifies each record

> 💡 The primary key uniquely identifies each row in a table. Like the ID card of each person!

### 5. 🔗 What is a foreign key? 🟡

A) 🔗 Field that references another table

B) 🔒 External access key

C) 🛡️ Security key

D) 🔐 Encryption key

**Correct Answer**: A) 🔗 Field that references another table

> 📘 Foreign keys create relationships between tables. Like a link that connects related information!

### 6. 🔍 What does the SELECT command do? 🟢

A) 🔍 Retrieve data from a table

B) 🔧 Select files

C) 📦 Choose packages

D) 🌐 Connect networks

**Correct Answer**: A) 🔍 Retrieve data from a table

> 🎯 `SELECT` allows querying and retrieving information from tables. Like asking a question to the database!

### 7. ➕ What does the INSERT command do? 🟢

A) ➕ Insert new records

B) 🔧 Install software

C) 📦 Import data

D) 🌐 Connect servers

**Correct Answer**: A) ➕ Insert new records

> ⚡ `INSERT` adds new data to a table. Like adding a new entry to your agenda!

### 8. ✏️ What does the UPDATE command do? 🟡

A) ✏️ Modify existing records

B) 🔧 Update software

C) 📦 Update files

D) 🌐 Update network

**Correct Answer**: A) ✏️ Modify existing records

> 💡 `UPDATE` allows changing existing data in tables. Like correcting information in a form!

### 9. 🗑️ What does the DELETE command do? 🟢

A) 🗑️ Delete records

B) 🔧 Delete files

C) 📦 Uninstall software

D) 🌐 Disconnect network

**Correct Answer**: A) 🗑️ Delete records

> 📘 `DELETE` removes records from a table. Like crossing out an entry from your list!

### 10. 🔍 What is WHERE in SQL? 🟢

A) 🔍 Condition to filter results

B) 🔧 File location

C) 📦 Package directory

D) 🌐 Network address

**Correct Answer**: A) 🔍 Condition to filter results

> 🎯 `WHERE` allows filtering data based on specific conditions. Like using a filter to find what you're looking for!

### 11. 📊 What is ORDER BY in SQL? 🟡

A) 📊 Sort results

B) 🔧 Organize files

C) 📦 Classify packages

D) 🌐 Sort connections

**Correct Answer**: A) 📊 Sort results

> ⚡ `ORDER BY` sorts the results of a query. Like organizing your library by author or title!

### 12. 🔗 What is JOIN in SQL? 🟡

A) 🔗 Combine data from multiple tables

B) 🔧 Connect servers

C) 📦 Join files

D) 🌐 Connect networks

**Correct Answer**: A) 🔗 Combine data from multiple tables

> 💡 `JOIN` allows combining information from different tables. Like putting together puzzle pieces!

### 13. 📊 What is GROUP BY in SQL? 🟡

A) 📊 Group results

B) 🔧 Organize files

C) 📦 Classify packages

D) 🌐 Group connections

**Correct Answer**: A) 📊 Group results

> 📘 `GROUP BY` groups similar records. Like organizing books by categories!

### 14. 🔍 What is an index in a database? 🟡

A) 🔍 Structure that improves search speed

B) 🔧 File list

C) 📦 Package catalog

D) 🌐 Network map

**Correct Answer**: A) 🔍 Structure that improves search speed

> 🎯 Indexes speed up queries like a book index. Like having bookmarks to find information quickly!

### 15. 🔐 What is a transaction in a database? 🟡

A) 🔐 Group of operations that execute as a unit

B) 🔒 Security operation

C) 🛡️ Encryption process

D) 🔑 Access key

**Correct Answer**: A) 🔐 Group of operations that execute as a unit

> ⚡ Transactions ensure that all operations complete or none do. Like transferring money: all or nothing!

### 16. 🔄 What is ACID in databases? 🟡

A) 🔄 Transaction properties (Atomicity, Consistency, Isolation, Durability)

B) 🔧 Database type

C) 📦 Communication protocol

D) 🌐 Network system

**Correct Answer**: A) 🔄 Transaction properties (Atomicity, Consistency, Isolation, Durability)

> 💡 ACID guarantees data integrity in transactions. Like security rules for your data!

### 17. 🔍 What is a query? 🟢

A) 🔍 Question to the database

B) 🔧 System command

C) 📦 Package request

D) 🌐 Network request

**Correct Answer**: A) 🔍 Question to the database

> 📘 A query is an instruction to retrieve or manipulate data. Like asking a specific question to the database!

### 18. 📊 What is a view in SQL? 🟡

A) 📊 Saved query as virtual table

B) 🔧 File view

C) 📦 Package view

D) 🌐 Network view

**Correct Answer**: A) 📊 Saved query as virtual table

> 🎯 Views are saved queries that behave like tables. Like having a personalized window to your data!

### 19. 🔐 What is a trigger in a database? 🟡

A) 🔐 Procedure that executes automatically

B) 🔒 Security trigger

C) 🛡️ Alert activator

D) 🔑 Activation key

**Correct Answer**: A) 🔐 Procedure that executes automatically

> ⚡ Triggers execute automatically when certain events occur. Like a sensor that reacts automatically!

### 20. 📊 What is normalization in databases? 🟡

A) 📊 Process of organizing data to reduce redundancy

B) 🔧 Normalize files

C) 📦 Standardize packages

D) 🌐 Normalize networks

**Correct Answer**: A) 📊 Process of organizing data to reduce redundancy

> 💡 Normalization eliminates duplicate data and improves integrity. Like organizing your closet to avoid duplicates!

### 21. 🔍 What is a stored procedure? 🟡

A) 🔍 Set of saved SQL instructions

B) 🔧 System procedure

C) 📦 Instruction package

D) 🌐 Network protocol

**Correct Answer**: A) 🔍 Set of saved SQL instructions

> 📘 Stored procedures are reusable SQL programs. Like having saved recipes for common tasks!
