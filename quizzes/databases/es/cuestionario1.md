# 🗄️ Databases - Cuestionario 1

## Preguntas

### 1. 🗄️ ¿Qué es una base de datos? 🟢

A) 🗄️ Sistema para almacenar y gestionar datos

B) 🔧 Servidor de archivos

C) 📦 Gestor de paquetes

D) 🌐 Sistema de red

**Respuesta correcta**: A) 🗄️ Sistema para almacenar y gestionar datos

> 💡 Una base de datos es un sistema organizado para almacenar, recuperar y gestionar información. ¡Como una biblioteca digital bien organizada!

### 2. 🔍 ¿Qué es SQL? 🟢

A) 🔍 Structured Query Language

B) 🔧 Sistema de archivos

C) 📦 Protocolo de red

D) 🌐 Lenguaje de programación

**Respuesta correcta**: A) 🔍 Structured Query Language

> 📘 SQL es el lenguaje estándar para interactuar con bases de datos relacionales. ¡Como el idioma universal para hablar con bases de datos!

### 3. 🔗 ¿Qué es una base de datos relacional? 🟢

A) 🔗 Base de datos que organiza datos en tablas relacionadas

B) 🔧 Base de datos en la nube

C) 📦 Base de datos de archivos

D) 🌐 Base de datos distribuida

**Respuesta correcta**: A) 🔗 Base de datos que organiza datos en tablas relacionadas

> ⚡ Las bases de datos relacionales usan tablas para organizar datos con relaciones entre ellas. ¡Como un sistema de carpetas bien organizado!

### 4. 🔑 ¿Qué es una clave primaria? 🟡

A) 🔑 Campo único que identifica cada registro

B) 🔒 Contraseña de acceso

C) 🛡️ Clave de encriptación

D) 🔐 Clave de seguridad

**Respuesta correcta**: A) 🔑 Campo único que identifica cada registro

> 💡 La clave primaria identifica únicamente cada fila en una tabla. ¡Como la cédula de identidad de cada persona!

### 5. 🔗 ¿Qué es una clave foránea? 🟡

A) 🔗 Campo que referencia otra tabla

B) 🔒 Clave de acceso externa

C) 🛡️ Clave de seguridad

D) 🔐 Clave de encriptación

**Respuesta correcta**: A) 🔗 Campo que referencia otra tabla

> 📘 Las claves foráneas crean relaciones entre tablas. ¡Como un enlace que conecta información relacionada!

### 6. 🔍 ¿Qué hace el comando SELECT? 🟢

A) 🔍 Recuperar datos de una tabla

B) 🔧 Seleccionar archivos

C) 📦 Elegir paquetes

D) 🌐 Conectar redes

**Respuesta correcta**: A) 🔍 Recuperar datos de una tabla

> 🎯 `SELECT` permite consultar y recuperar información de las tablas. ¡Como hacer una pregunta a la base de datos!

### 7. ➕ ¿Qué hace el comando INSERT? 🟢

A) ➕ Insertar nuevos registros

B) 🔧 Instalar software

C) 📦 Importar datos

D) 🌐 Conectar servidores

**Respuesta correcta**: A) ➕ Insertar nuevos registros

> ⚡ `INSERT` añade nuevos datos a una tabla. ¡Como agregar una nueva entrada a tu agenda!

### 8. ✏️ ¿Qué hace el comando UPDATE? 🟡

A) ✏️ Modificar registros existentes

B) 🔧 Actualizar software

C) 📦 Actualizar archivos

D) 🌐 Actualizar red

**Respuesta correcta**: A) ✏️ Modificar registros existentes

> 💡 `UPDATE` permite cambiar datos existentes en las tablas. ¡Como corregir información en un formulario!

### 9. 🗑️ ¿Qué hace el comando DELETE? 🟢

A) 🗑️ Eliminar registros

B) 🔧 Eliminar archivos

C) 📦 Desinstalar software

D) 🌐 Desconectar red

**Respuesta correcta**: A) 🗑️ Eliminar registros

> 📘 `DELETE` elimina registros de una tabla. ¡Como tachar una entrada de tu lista!

### 10. 🔍 ¿Qué es WHERE en SQL? 🟢

A) 🔍 Condición para filtrar resultados

B) 🔧 Ubicación de archivo

C) 📦 Directorio de paquetes

D) 🌐 Dirección de red

**Respuesta correcta**: A) 🔍 Condición para filtrar resultados

> 🎯 `WHERE` permite filtrar datos basándose en condiciones específicas. ¡Como usar un filtro para encontrar lo que buscas!

### 11. 📊 ¿Qué es ORDER BY en SQL? 🟡

A) 📊 Ordenar resultados

B) 🔧 Organizar archivos

C) 📦 Clasificar paquetes

D) 🌐 Ordenar conexiones

**Respuesta correcta**: A) 📊 Ordenar resultados

> ⚡ `ORDER BY` ordena los resultados de una consulta. ¡Como organizar tu biblioteca por autor o título!

### 12. 🔗 ¿Qué es JOIN en SQL? 🟡

A) 🔗 Combinar datos de múltiples tablas

B) 🔧 Conectar servidores

C) 📦 Unir archivos

D) 🌐 Conectar redes

**Respuesta correcta**: A) 🔗 Combinar datos de múltiples tablas

> 💡 `JOIN` permite combinar información de diferentes tablas. ¡Como juntar piezas de un rompecabezas!

### 13. 📊 ¿Qué es GROUP BY en SQL? 🟡

A) 📊 Agrupar resultados

B) 🔧 Organizar archivos

C) 📦 Clasificar paquetes

D) 🌐 Agrupar conexiones

**Respuesta correcta**: A) 📊 Agrupar resultados

> 📘 `GROUP BY` agrupa registros similares. ¡Como organizar libros por categorías!

### 14. 🔍 ¿Qué es un índice en una base de datos? 🟡

A) 🔍 Estructura que mejora la velocidad de búsqueda

B) 🔧 Lista de archivos

C) 📦 Catálogo de paquetes

D) 🌐 Mapa de red

**Respuesta correcta**: A) 🔍 Estructura que mejora la velocidad de búsqueda

> 🎯 Los índices aceleran las consultas como un índice de libro. ¡Como tener marcadores para encontrar información rápidamente!

### 15. 🔐 ¿Qué es una transacción en una base de datos? 🟡

A) 🔐 Grupo de operaciones que se ejecutan como una unidad

B) 🔒 Operación de seguridad

C) 🛡️ Proceso de encriptación

D) 🔑 Clave de acceso

**Respuesta correcta**: A) 🔐 Grupo de operaciones que se ejecutan como una unidad

> ⚡ Las transacciones aseguran que todas las operaciones se completen o ninguna. ¡Como transferir dinero: todo o nada!

### 16. 🔄 ¿Qué es ACID en bases de datos? 🟡

A) 🔄 Propiedades de transacciones (Atomicidad, Consistencia, Aislamiento, Durabilidad)

B) 🔧 Tipo de base de datos

C) 📦 Protocolo de comunicación

D) 🌐 Sistema de red

**Respuesta correcta**: A) 🔄 Propiedades de transacciones (Atomicidad, Consistencia, Aislamiento, Durabilidad)

> 💡 ACID garantiza la integridad de los datos en las transacciones. ¡Como reglas de seguridad para tus datos!

### 17. 🔍 ¿Qué es una consulta? 🟢

A) 🔍 Pregunta a la base de datos

B) 🔧 Comando del sistema

C) 📦 Solicitud de paquete

D) 🌐 Petición de red

**Respuesta correcta**: A) 🔍 Pregunta a la base de datos

> 📘 Una consulta es una instrucción para recuperar o manipular datos. ¡Como hacer una pregunta específica a la base de datos!

### 18. 📊 ¿Qué es una vista en SQL? 🟡

A) 📊 Consulta guardada como tabla virtual

B) 🔧 Vista de archivo

C) 📦 Vista de paquete

D) 🌐 Vista de red

**Respuesta correcta**: A) 📊 Consulta guardada como tabla virtual

> 🎯 Las vistas son consultas guardadas que se comportan como tablas. ¡Como tener una ventana personalizada a tus datos!

### 19. 🔐 ¿Qué es un trigger en una base de datos? 🟡

A) 🔐 Procedimiento que se ejecuta automáticamente

B) 🔒 Activador de seguridad

C) 🛡️ Activador de alerta

D) 🔑 Clave de activación

**Respuesta correcta**: A) 🔐 Procedimiento que se ejecuta automáticamente

> ⚡ Los triggers se ejecutan automáticamente cuando ocurren ciertos eventos. ¡Como un sensor que reacciona automáticamente!

### 20. 📊 ¿Qué es la normalización en bases de datos? 🟡

A) 📊 Proceso de organizar datos para reducir redundancia

B) 🔧 Normalizar archivos

C) 📦 Estandarizar paquetes

D) 🌐 Normalizar redes

**Respuesta correcta**: A) 📊 Proceso de organizar datos para reducir redundancia

> 💡 La normalización elimina datos duplicados y mejora la integridad. ¡Como organizar tu armario para evitar duplicados!

### 21. 🔍 ¿Qué es un procedimiento almacenado? 🟡

A) 🔍 Conjunto de instrucciones SQL guardadas

B) 🔧 Procedimiento del sistema

C) 📦 Paquete de instrucciones

D) 🌐 Protocolo de red

**Respuesta correcta**: A) 🔍 Conjunto de instrucciones SQL guardadas

> 📘 Los procedimientos almacenados son programas SQL reutilizables. ¡Como tener recetas guardadas para tareas comunes!
