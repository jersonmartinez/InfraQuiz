# 📊 Monitoring - Cuestionario 1

## Preguntas

### 1. 📊 ¿Qué es el monitoreo de sistemas? 🟢

A) 📊 Observar y medir el rendimiento del sistema

B) 🔧 Configurar servidores

C) 📦 Instalar software

D) 🌐 Conectar redes

**Respuesta correcta**: A) 📊 Observar y medir el rendimiento del sistema

> 💡 El monitoreo permite detectar problemas y optimizar el rendimiento del sistema. ¡Como tener un panel de control para tu infraestructura!

### 2. 📈 ¿Qué son las métricas? 🟢

A) 📈 Mediciones cuantitativas del sistema

B) 🔧 Configuraciones del sistema

C) 📦 Datos de rendimiento

D) 🌐 Estadísticas de red

**Respuesta correcta**: A) 📈 Mediciones cuantitativas del sistema

> 📘 Las métricas son números que indican el estado del sistema (CPU, memoria, etc.). ¡Como el velocímetro de tu coche!

### 3. 🔔 ¿Qué es una alerta? 🟡

A) 🔔 Notificación cuando algo va mal

B) 🔧 Mensaje del sistema

C) 📦 Advertencia de paquete

D) 🌐 Alarma de red

**Respuesta correcta**: A) 🔔 Notificación cuando algo va mal

> ⚡ Las alertas te notifican cuando hay problemas que necesitan atención. ¡Como un detector de humo para tus sistemas!

### 4. 📊 ¿Qué es un dashboard? 🟢

A) 📊 Interfaz visual para mostrar métricas

B) 🔧 Panel de control

C) 📦 Vista de datos

D) 🌐 Monitor de red

**Respuesta correcta**: A) 📊 Interfaz visual para mostrar métricas

> 💡 Los dashboards muestran información importante de forma visual y fácil de entender. ¡Como un GPS para tu infraestructura!

### 5. 🔍 ¿Qué es el logging? 🟢

A) 🔍 Registrar eventos del sistema

B) 🔧 Archivo de configuración

C) 📦 Log de paquetes

D) 🌐 Log de red

**Respuesta correcta**: A) 🔍 Registrar eventos del sistema

> 🎯 Los logs registran todo lo que pasa en el sistema para investigar problemas. ¡Como un diario de tu aplicación!

### 6. 📊 ¿Qué es Prometheus? 🟡

A) 📊 Sistema de monitoreo y alertas

B) 🔧 Herramienta de configuración

C) 📦 Gestor de métricas

D) 🌐 Servidor de monitoreo

**Respuesta correcta**: A) 📊 Sistema de monitoreo y alertas

> ⚡ Prometheus recopila y almacena métricas en formato de series temporales. ¡La solución de monitoreo más popular en el mundo cloud-native!

### 7. 📊 ¿Qué es Grafana? 🟡

A) 📊 Plataforma de visualización y análisis

B) 🔧 Recolector de datos

C) 📦 Almacenamiento de métricas

D) 🌐 Agente de monitoreo

**Respuesta correcta**: A) 📊 Plataforma de visualización y análisis

> 📘 Grafana crea dashboards hermosos para visualizar tus datos. ¡Como un artista que convierte números en imágenes bonitas!

### 8. 🔔 ¿Qué es Alertmanager? 🟡

A) 🔔 Maneja alertas enviadas por Prometheus

B) 🔧 Crea alertas

C) 📦 Almacena alertas

D) 🌐 Envía métricas

**Respuesta correcta**: A) 🔔 Maneja alertas enviadas por Prometheus

> 💡 Alertmanager gestiona alertas: las agrupa, las enruta y envía notificaciones. ¡Como un sistema de correo inteligente para tus alertas!

### 9. 📊 ¿Qué es un SLA? 🟡

A) 📊 Service Level Agreement

B) 🔧 System Level Analysis

C) 📦 Server Level Automation

D) 🌐 Service Level Alert

**Respuesta correcta**: A) 📊 Service Level Agreement

> 🎯 SLA define el nivel de servicio esperado. ¡Como un contrato que promete qué tan bien funcionará tu sistema!

### 10. 📈 ¿Qué es un SLI? 🟡

A) 📈 Service Level Indicator

B) 🔧 System Level Interface

C) 📦 Server Level Index

D) 🌐 Service Level Integration

**Respuesta correcta**: A) 📈 Service Level Indicator

> ⚡ SLI es una métrica específica que mide el rendimiento del servicio. ¡Como las mediciones reales que muestran si cumples tus promesas!

### 11. 🎯 ¿Qué es un SLO? 🟡

A) 🎯 Service Level Objective

B) 🔧 System Level Operation

C) 📦 Server Level Organization

D) 🌐 Service Level Optimization

**Respuesta correcta**: A) 🎯 Service Level Objective

> 📘 SLO es el valor objetivo para SLI. ¡Como la meta que quieres alcanzar!

### 12. 📊 ¿Qué es uptime? 🟢

A) 📊 Tiempo cuando el sistema está disponible y funcionando

B) 🔧 Tiempo de inicio del sistema

C) 📦 Tiempo de actualización

D) 🌐 Tiempo de conexión de red

**Respuesta correcta**: A) 📊 Tiempo cuando el sistema está disponible y funcionando

> 💡 Uptime mide la disponibilidad del sistema. ¡Usualmente se expresa como porcentaje (99.9% uptime = ¡muy bueno!)

### 13. 🔍 ¿Qué es un agregador de logs? 🟡

A) 🔍 Sistema que recopila logs de múltiples fuentes

B) 🔧 Archivo de log

C) 📦 Formato de log

D) 🌐 Visor de logs

**Respuesta correcta**: A) 🔍 Sistema que recopila logs de múltiples fuentes

> 🎯 Los agregadores de logs como ELK Stack recopilan logs de todas partes en un lugar. ¡Como tener todos tus libros en una biblioteca!

### 14. 📈 ¿Qué es el monitoreo sintético? 🟡

A) 📈 Pruebas simuladas para verificar el comportamiento del sistema

B) 🔧 Monitoreo de usuarios reales

C) 📦 Monitoreo natural

D) 🌐 Monitoreo manual

**Respuesta correcta**: A) 📈 Pruebas simuladas para verificar el comportamiento del sistema

> ⚡ El monitoreo sintético ejecuta pruebas automatizadas para detectar problemas antes que los usuarios. ¡Como tener un robot cliente probando tu tienda!

### 15. 👥 ¿Qué es RUM? 🟡

A) 👥 Real User Monitoring

B) 🔧 Resource Usage Monitoring

C) 📦 Runtime Usage Metrics

D) 🌐 Remote User Management

**Respuesta correcta**: A) 👥 Real User Monitoring

> 📘 RUM mide la experiencia real del usuario. ¡Como tener un espía que observa cómo tus clientes realmente usan tu aplicación!

### 16. ⚠️ ¿Qué es MTTR? 🟡

A) ⚠️ Mean Time To Repair/Recovery

B) 🔧 Maximum Time To Respond

C) 📦 Minimum Time To Restart

D) 🌐 Manual Time To Review

**Respuesta correcta**: A) ⚠️ Mean Time To Repair/Recovery

> 💡 MTTR mide qué tan rápido arreglas problemas. ¡Menor es mejor - como ser un mecánico rápido!

### 17. 💥 ¿Qué es MTBF? 🟡

A) 💥 Mean Time Between Failures

B) 🔧 Maximum Time Before Failure

C) 📦 Minimum Time Between Fixes

D) 🌐 Manual Time Before Freeze

**Respuesta correcta**: A) 💥 Mean Time Between Failures

> 🎯 MTBF mide qué tan seguido se rompen las cosas. ¡Mayor es mejor - como tener un coche confiable!

### 18. 📊 ¿Qué es una base de datos de series temporales? 🟡

A) 📊 Base de datos optimizada para datos con marca temporal

B) 🔧 Base de datos para almacenar tiempo

C) 📦 Base de datos con funciones de tiempo

D) 🌐 Base de datos para programación

**Respuesta correcta**: A) 📊 Base de datos optimizada para datos con marca temporal

> ⚡ Las bases de datos de series temporales como InfluxDB son perfectas para almacenar métricas a lo largo del tiempo. ¡Como un archivero especializado para datos históricos!

### 19. 🔍 ¿Qué es APM? 🟡

A) 🔍 Application Performance Monitoring

B) 🔧 Automated Process Management

C) 📦 Application Package Manager

D) 🌐 Advanced Performance Metrics

**Respuesta correcta**: A) 🔍 Application Performance Monitoring

> 📘 APM monitorea el rendimiento de aplicaciones y la experiencia del usuario. ¡Como tener un entrenador de rendimiento para tu aplicación!

### 20. 📊 ¿Qué es el rastreo distribuido? 🔴

A) 📊 Rastrear solicitudes a través de múltiples servicios

B) 🔧 Rastrear conexiones de red

C) 📦 Rastrear distribuciones de archivos

D) 🌐 Rastrear rutas de usuarios

**Respuesta correcta**: A) 📊 Rastrear solicitudes a través de múltiples servicios

> 💡 El rastreo distribuido sigue solicitudes a través de microservicios. ¡Como seguir una carta a través del sistema postal!

### 21. 📈 ¿Cuáles son las señales doradas? 🔴

A) 📈 Latencia, tráfico, errores y saturación

B) 🔧 CPU, memoria, disco y red

C) 📦 Arriba, abajo, lento y rápido

D) 🌐 Iniciar, parar, pausar y reanudar

**Respuesta correcta**: A) 📈 Latencia, tráfico, errores y saturación

> 🏆 Las cuatro señales doradas son las métricas más importantes para monitorear. ¡Como los signos vitales para la salud de tu sistema!
