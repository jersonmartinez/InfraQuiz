# ☁️ AWS - Questions 1

## Questions

### 1. 🏢 What is Amazon Web Services (AWS)? 🟢

A) ☁️ `Cloud computing platform`
B) 🔧 `Operating system`
C) ⚙️ `Local database`
D) 📦 `Physical server`

**Correct Answer**: A) ☁️ `Cloud computing platform`

> 💡 AWS is the world's largest cloud computing platform. Like having a giant supercomputer available 24/7!

### 2. 🖥️ What is Amazon EC2? 🟢

A) ⚡ `Virtual server service`
B) 🔧 `Database`
C) 📦 `File storage`
D) 🌐 `Network service`

**Correct Answer**: A) ⚡ `Virtual server service`

> 📘 EC2 allows you to create and manage virtual servers in the cloud. Like renting computers on the internet!

### 3. 💾 What is Amazon S3? 🟢

A) 📦 `Object storage service`
B) 🗄️ `Relational database`
C) 🔧 `Web server`
D) ⚙️ `Compute service`

**Correct Answer**: A) 📦 `Object storage service`

> 🎯 S3 is AWS's most popular object storage service. Like a gigantic digital closet for your files!
